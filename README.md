# 🧠 Neural Grid – Logic Circuit Strategy Game

Neural Grid is a 2048-inspired strategy game built using Python and Flask.  
Instead of numbers, the game uses logic gates (AND, OR, XOR) that merge and level up.

---

## 🎮 How to Play

- The board is a 4x4 grid.
- Each tile represents a logic gate.
- Tiles merge only if:
  - Same gate type
  - Same level
- After every move, a new tile appears.
- Reach **Level 5** of any gate to win.
- If no moves are possible, the game ends.
